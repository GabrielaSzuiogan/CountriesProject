# CountriesProject
 A simple web application that allows users to search for information about countries around the world.

## Features

- Search for any country by name
- View country details including:
  - Flag image
  - Capital city
  - Languages spoken
  - Population
  - Currency information
  - Link to Google Maps

## How to Start

1. **Open the project** - Open the project folder in Visual Studio Code

2. **Start Live Server** - Right-click on `index.html` and select "Open with Live Server"

3. **Search for a country** - Type a country name in the search input field

4. **Click Search** - Click the "Search" button to display country information

5. **View results** - The country's details will appear below with its flag, capital, languages, population, currency, and a link to its location on Google Maps
